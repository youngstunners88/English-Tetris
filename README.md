# English Tetris

> Learn English vocabulary while playing Tetris. Words fall like blocks — type them before they stack up!

## Concept

English Tetris combines the classic falling block gameplay with language learning. Instead of geometric shapes, words fall from the top of the screen. Players must type the words correctly before they reach the bottom.

## Features

- **Falling Words** — Vocabulary words descend at increasing speeds
- **Difficulty Levels** — Easy (common words) → Hard (advanced vocabulary)
- **Categories** — Business, Travel, Academic, Casual, Technical
- **Scoring** — Speed + accuracy + word complexity
- **Progress Tracking** — Learned words, speed improvements, streaks

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Game Engine:** Custom React hooks with requestAnimationFrame
- **State Management:** Zustand
- **Backend (optional):** Hono API for leaderboards & word banks

## Quick Start

```bash
# Clone and install
git clone https://github.com/youngstunners88/English-Tetris.git
cd English-Tetris
npm install

# Start development
npm run dev
```

## Game Modes

1. **Classic** — Standard Tetris with typing words to clear lines
2. **Survival** — Endless mode, words get faster over time
3. **Challenge** — Specific categories (IELTS, Business English, etc.)
4. **Multiplayer** — Race against others in real-time
